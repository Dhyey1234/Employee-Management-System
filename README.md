# Employee Management System

A full-stack web application to manage employee records, including filtering, data management, and retirement tracking.

## 🚀 Features
- Add, update, and delete employee records (CRUD)
- Filter and search employees
- Manage employee lifecycle and retirement data
- Responsive user interface

## 🛠 Tech Stack
- Frontend: React, HTML, CSS
- Backend: Node.js, Express.js
- API: REST & GraphQL
- Database: MongoDB
- Tools: Git

## ⚙️ Functionality
- Built scalable APIs for handling employee data
- Implemented efficient database queries
- Performed debugging and testing for stability
- Followed clean code and modular structure

## 📌 Purpose
This project demonstrates full-stack development skills, including API design, database management, and responsive UI development.
